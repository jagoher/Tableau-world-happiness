# 🌍 World Happiness Dashboard

An interactive Tableau dashboard based on the World Happiness Report.

This project explores global happiness through interactive visualizations, highlighting the relationship between happiness, GDP per capita, social support, geographic regions, and historical trends.

---

## 📊 Dashboard Features

- 🌍 Interactive world choropleth map
- 📈 GDP per Capita vs Happiness scatter plot
- 📅 Happiness evolution over time
- 🌎 Average happiness by continent
- 🇳🇴 Nordic countries highlighted on the map
- 🎯 Interactive filters by year
- 📌 Tableau Groups, Sets and Hierarchies
- 💡 Dashboard Actions and Viz in Tooltip
